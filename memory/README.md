# Memory - Project History & Context

**Purpose:** Store session context, decisions, and learnings for long-term recall.

---

## Structure

| File | Description |
|------|-------------|
| `sessions/` | Session summaries and context snapshots |
| `decisions/` | Key architectural and design decisions |
| `learnings/` | Lessons learned, gotchas, and best practices |

---

## Usage

Use the `archivist-memory` skill to:
- Save context before ending a session
- Restore context at the start of a new session
- Track what was done and why
